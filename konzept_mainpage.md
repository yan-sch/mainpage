# Konzept: Persönliche Browser-Startseite (`konzept_mainpage`)

## Zielbild

Eine individuell gestaltete Browser-Startseite als persönliche Kommandozentrale für:

- Private Internetnutzung
- Schnellen Zugriff auf häufig genutzte Seiten
- Coding / Webentwicklung
- KI-Themen & Recherche
- Fokusarbeit und Produktivität
- Kontinuierliches Micro-Learning

Hosting über **GitHub Pages** als statische Website mit einfacher Aktualisierung per Git Push.

---

## Kernfunktionen (MVP)

## 1. Wetter-Widget

Anzeige von:

- Aktuelles Wetter
- Temperatur
- Standortbezogene Vorhersage
- Optional: 3-Tage-Ausblick

## 2. News-Widgets

Zwei kompakte News-Bereiche mit Headlines und Direktlinks:

### Wirtschaft

- Märkte
- Unternehmen
- Makroökonomie

### Coding / AI

- Programmierung
- Webentwicklung
- KI-News
- Neue Tools / Releases

Optional später:

- Kategorien dynamisch auswählbar
- RSS / API Quellen konfigurierbar

## 3. Favoriten-Dashboard

Moderne Darstellung häufig genutzter Seiten statt klassischer Browser-Favoritenleiste.

Beispiele:

- Mail
- GitHub
- ChatGPT
- YouTube
- Banking
- Tools
- Eigene Projekte

Darstellung:

- Große klickbare Icons
- Kategorien
- Suchfunktion

---

## Produktivitätsfunktionen

## 4. Pomodoro / Fokus-Timer

- 25 / 5 Minuten Standard
- Start / Pause / Reset
- Optional Sound
- Tagesstatistik

## 5. Bookmark / Leseliste

Schnelles Speichern interessanter Links für später.

Funktionen:

- URL speichern
- Titel automatisch übernehmen
- Checkbox „gelesen“
- Später öffnen

## 6. Mini To-do Bereich

Kleine Aufgabenliste für spontane Tasks.

---

## Lern- & Entwicklungsfunktionen

## 7. Daily Quiz Widget (Micro-Learning)

Bei jedem Öffnen erscheint optional eine Multiple-Choice-Frage zu:

- Coding
- Webentwicklung
- KI
- Tools
- Allgemeines Tech-Wissen

Ablauf:

1. Frage anzeigen
2. Antwort auswählen
3. Sofortiges Feedback
4. Kurze Erklärung

Nutzen:

- Täglicher Mini-Lerneffekt
- Wissensauffrischung
- Motivation

## 8. Code-Snippet Box (später)

Speicher für häufig genutzte Befehle / Snippets:

- Git Commands
- CSS Tricks
- JS Patterns
- Terminal Befehle
